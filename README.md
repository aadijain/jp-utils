# jp-utils

Japanese learning tooling in two halves: a reusable **backend service** for Japanese text processing that keeps a personal vocabulary store, and a thin **Anki add-on** that uses it to enrich cards and reorder decks. The backend speaks plain HTTP, so other tools can build on it too.
